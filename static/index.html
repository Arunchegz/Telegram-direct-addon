Hybrid Streaming Architecture (4 paths):
  • Path A (local) — Byte-range exists on disk. Served instantly from local SparseFile cache.
  • Path B (local-waited) — Byte-range is currently downloading. Thread waits up to 2s for download to land.
  • Path C (mixed) — Serves already-cached prefix locally, and transparently shifts back to Telegram live download for remainder.
  • Path D (telegram-live) — Direct streaming fallback via Telegram MTProto.

Delete Media Item

This will remove the media file from your Stremio library and permanently erase any local cached files from disk.
